Virtual Library API

REST API for virtual library management, with semantic search powered by OpenRouter embeddings.

📚 API Docs (Scalar) OpenAPI Spec Health

Features

  • Full CRUD for books with filtering, pagination and ordering
  • LLM-generated summary on create (optional)
  • Semantic search via bge-m3 embeddings
  • RFC 7807 Problem Details for errors
  • Structured logging with request IDs
